## Réservation Google Calendar
1. Ouvre Google Calendar sur ordinateur.
2. Crée un Planning de rendez-vous / Appointment schedule.
3. Copie le lien de réservation.
4. Ouvre `script.js`.
5. Remplace :
   PASTE_YOUR_GOOGLE_CALENDAR_BOOKING_LINK_HERE
   par ton vrai lien.

## Chatbot intégré
Le site contient un assistant gratuit en JavaScript.
Il répond aux questions simples sur :
- rendez-vous
- tarifs
- électroménager
- TV / consoles
- PC / antivirus
- téléphone / tablette
- Wi‑Fi / câbles
- moyens de paiement

Aucun abonnement n’est nécessaire.


## Chatbot V6 amélioré
Le chatbot utilise maintenant une détection par synonymes et mots-clés étendus.
Il reconnaît aussi le type d’intervention souhaité :
- réparation / dépannage
- diagnostic
- installation / mise en service
- entretien / nettoyage / optimisation
- transfert de données
